OpenStreetMap (OSM) is a community-led mapping project providing free geographic data to anyone who wants to use them.

{{ site_name }} uses OSM data under the Creative Commons Attribution-Share Alike (CC-BY-SA) 2.0 license to provide maps and points-of-interest data.

Should you wish to contribute to OSM, you can either submit corrections though {{ site_name }} (there's a Suggest a correction link at the bottom of each OSM-derived page), or set up an OSM account and edit directly. For more information on getting started, see the Beginners' Guide (link below) on the OpenStreetMap website.

Here are some useful links you may want to browse from a non-mobile browser:

  1. Creative Commons Attribution-Share Alike 2.0 license
  2. The OpenStreetMap website
  3. The OpenStreetMap wiki
  4. The OSM Beginners' Guide
