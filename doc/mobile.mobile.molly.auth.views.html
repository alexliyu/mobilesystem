SecureView

An 'abstract' view for all secure pages.

Implements a timeout so that people will need to reauthenticate if a secure page hasn't been accessed recently. The SecureMidddleware checks for this class in a view's MRO in order to enforce an HTTPS connection.

TimedOutView

Handles the case where a user's secure session has been inactive for too long.

This view should not be included in a urlconf as it is referenced only by SecureView.
