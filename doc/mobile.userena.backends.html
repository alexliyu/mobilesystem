UserenaAuthenticationBackend.authenticate

authenticate(identification, password=None, check_password=True)

Authenticates a user through the combination email/username with password.

:param identification: A string containing the username or e-mail of the user that is trying to authenticate.

:password: Optional string containing the password for the user.

:param check_password: Boolean that defines if the password should be checked for this user. Always keep this ``True``. This is only used by userena at activation when a user opens a page with a secret hash.

:return: The signed in :class:`User`.
