SignupFormOnlyEmail

Form for creating a new user account but not needing a username.

This form is an adaptation of :class:`SignupForm`. It's used when ``USERENA_WITHOUT_USERNAME`` setting is set to ``True``. And thus the user is not asked to supply an username, but one is generated for them. The user can than keep sign in by using their email.
